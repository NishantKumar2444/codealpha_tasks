# Phishing Awareness Training

## Overview

This project was completed as part of the **CodeAlpha Cyber Security Internship**.

The objective of this project is to educate users about phishing attacks, how they work, how to recognize them, and the best practices to prevent becoming a victim.

---

## Project Contents

- `Phishing_Awareness_Training.pptx` – PowerPoint presentation.

---

## Topics Covered

- Understanding Phishing
- Common Types of Phishing
- Recognizing Phishing Emails
- Identifying Fake Websites
- Social Engineering Techniques
- Real-World Phishing Examples
- Best Security Practices
- What to Do if You Suspect an Attack
- Interactive Quiz

---

## Learning Objectives

After completing this presentation, viewers should be able to:

- Understand what phishing is.
- Recognize common phishing techniques.
- Identify suspicious emails and fake websites.
- Understand social engineering tactics used by attackers.
- Follow cybersecurity best practices to protect personal and organizational data.
